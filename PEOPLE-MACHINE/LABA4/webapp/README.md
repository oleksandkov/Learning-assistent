# LABA4 · Web prototype (React + SQLite)

Цей проєкт реалізує веб-інтерфейс для сценаріїв ЛР3 (розклад занять) у форматі ЛР4.

## Що реалізовано

- React UI (Vite)
- Локальна SQL база даних SQLite (`server/data/schedule.db`)
- Зберігання користувачів (логін + хеш пароля)
- Авторизація через JWT
- Довідники: предмети, групи, викладачі
- Розклад занять: перегляд, фільтрація, додавання, редагування, видалення
- Подвійний клік по рядку = редагування

## Запуск

1. Встановити залежності:
   - `npm install`
2. Запустити клієнт + сервер:
   - `npm run dev`
3. Відкрити:
   - `http://localhost:5173`

## Змінні оточення

Файл `.env` вже створений. Для реального використання змініть:

- `JWT_SECRET=CHANGE_ME_TO_A_LONG_RANDOM_SECRET`

## Корисно знати

- API працює на порту `4000`
- Frontend проксить `/api` на бекенд через Vite proxy
- База створюється автоматично при першому запуску
