# LABA3 (C++ / Windows Forms)

Це реалізація лабораторної №3 у стилі прикладу **BookshelfApp**, але на **C++ (C++/CLI, Windows Forms)**.

## Реалізовано

- завантаження книг з XML;
- показ книг у `DataGridView`;
- показ авторів вибраної книги у другій таблиці;
- контекстне меню для таблиці книг:
  - `addRowToolStripMenuItem` — додати рядок;
  - `removeRowToolStripMenuItem` — видалити рядок;
- сортування книг за назвою;
- пошук книг за прізвищем автора (результат у окремому вікні);
- збереження даних у XML.

## Де код

- `laba3.cpp` — весь код застосунку (моделі + форми + `main`).

## Як запустити у Visual Studio

1. Створіть проект: **CLR Empty Project (.NET Framework)**.
2. Додайте файл `laba3.cpp` у проект.
3. У властивостях проекту переконайтесь, що увімкнено **/clr**.
4. Зберіть і запустіть.

> Важливо: це саме WinForms на C++/CLI (не стандартний native C++ WinAPI).
