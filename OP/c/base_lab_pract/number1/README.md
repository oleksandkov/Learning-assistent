# printf()
**З якого заголовочного файлу:** stdio.h
**Що робить:** Виводить текст і значення змінних на екран.
**Як використовувати:**
```c
printf("Текст %d", змінна);
```
**Приклад:**
```c
#include <stdio.h>
int main() {
	int x = 5;
	printf("Значення: %d\n", x);
	return 0;
}
```

# if / else
**З якого заголовочного файлу:** не потребує заголовку
**Що робить:** Виконує різні дії залежно від умови.
**Як використовувати:**
```c
if (умова) {
	// дії
} else {
	// інші дії
}
```
**Приклад:**
```c
int main() {
	int x = 5;
	if (x > 0) {
		printf("Додатнє\n");
	} else {
		printf("Недодатнє\n");
	}
	return 0;
}
```

# double, int
**З якого заголовочного файлу:** не потребує заголовку
**Що робить:** double — дійсне число, int — ціле число.
**Як використовувати:**
```c
double a = 3.14;
int b = 5;
```
**Приклад:**
```c
int main() {
	double a = 2.5;
	int b = 3;
	printf("a = %.2lf, b = %d\n", a, b);
	return 0;
}
```

# sin(), cos()

# hypot()
**З якого заголовочного файлу:** math.h
**Що робить:** Обчислює довжину сторони трикутника за координатами.
**Як використовувати:**
```c

	return 0;

```

# stdio.h
Це стандартна бібліотека вводу/виводу в мові C (input/output stream).
Вона містить функції для роботи з текстовими потоками, наприклад, для зчитування даних з клавіатури та виведення на екран.

## Що робить stdio.h?
Дозволяє використовувати функції такі як printf(), scanf(), getchar(), putchar() та інші для роботи з введенням і виведенням.

## Як використовувати?
Потрібно додати на початку програми:
```c
#include <stdio.h>
```

## Приклад використання
```c
#include <stdio.h>

int main() {
	int x;
	printf("Введіть число: ");
	scanf("%d", &x);
	printf("Ви ввели: %d\n", x);
	return 0;
}
```

# Опис основних функцій та конструкцій, використаних у файлах task1.c - task23.c

## printf()
Функція для виведення тексту та значень змінних на екран. Формат: printf("текст %d", змінна);

## scanf()
Функція для зчитування значень з клавіатури. Формат: scanf("%d", &змінна);

## return
Оператор, який завершує виконання функції та повертає значення (у main зазвичай return 0; означає успішне завершення).

## if / else
Умовна конструкція для виконання різних дій залежно від умови. if (умова) { ... } else { ... }

## double, int
Типи змінних: double — дійсне число, int — ціле число.

## fabs()
Функція з math.h, повертає модуль (абсолютне значення) числа.

## sqrt()
Функція з math.h, обчислює квадратний корінь.

## pow()
Функція з math.h, піднесення до степеня: pow(число, степінь).

## sin(), cos()
Тригонометричні функції з math.h для обчислення синуса та косинуса.

## hypot()
Функція з math.h, обчислює довжину сторони трикутника за координатами: hypot(x2-x1, y2-y1).

## scanf("%lf", &a)
Зчитування дійсного числа (double) з клавіатури.

## scanf("%d", &a)
Зчитування цілого числа (int) з клавіатури.

## const
Оголошення константи, наприклад: const double PI = 3.14159;

## Математичні операції
Операції +, -, *, /, % для додавання, віднімання, множення, ділення та остачі від ділення.

## Коментарі
Однорядкові (//) та багаторядкові (/* ... */) коментарі для пояснення коду.
