# Lecture 2 - Classes and Objects

## Topics Covered
- Understanding Classes
- Creating and Using Objects
- Constructors and Methods
- Object State and Behavior
- Multiple Object Instances

## Examples
The `Main.java` file demonstrates:
- Creating a `Car` class with properties (brand, model, year)
- Instantiating multiple objects
- Using object methods (displayInfo, startEngine, stopEngine)
- Managing object state (engineRunning)

## How to Run

### IntelliJ IDEA:
1. Open this folder in IntelliJ IDEA
2. Open `src/Main.java`
3. Click the green play button or press `Shift + F10`

### VS Code:
1. Open `src/Main.java`
2. Press `Ctrl + Shift + B`
3. Select "Run Java Project (IntelliJ-style)"
