## 📝 Загальні вимоги до лабораторної роботи

### 1️⃣ Створення класу

**Необхідно:**
- Створити клас з **мінімум 5 полями** (атрибутами)
- Поля повинні бути **різних типів** (String, int, double, boolean тощо)
- Всі поля мають бути **приватними** (private)

**Приклад:**
```java
class Student {
    private String name;
    private String surname;
    private int course;
    private double averageGrade;
    private String faculty;
}
```

---

### 2️⃣ Конструктор

**Необхідно:**
- Створити **конструктор з параметрами** для ініціалізації всіх полів
- Використовувати ключове слово `this` для розрізнення полів і параметрів

**Приклад:**
```java
public Student(String name, String surname, int course, double averageGrade, String faculty) {
    this.name = name;
    this.surname = surname;
    this.course = course;
    this.averageGrade = averageGrade;
    this.faculty = faculty;
}
```

---

### 3️⃣ Геттери (Getters)

**Необхідно:**
- Створити **геттер для кожного поля**
- Назва геттера: `get` + назва поля з великої літери
- Геттер повертає значення поля

**Приклад:**
```java
public String getName() {
    return name;
}

public int getCourse() {
    return course;
}
```

---

### 4️⃣ Сеттери (Setters)

**Необхідно:**
- Створити **сеттер для кожного поля**
- Назва сеттера: `set` + назва поля з великої літери
- **Додати валідацію** (перевірку коректності даних)

**Приклад:**
```java
public void setCourse(int course) {
    if (course >= 1 && course <= 6) {
        this.course = course;
    } else {
        System.out.println("Помилка: курс має бути від 1 до 6");
    }
}
```

---

### 5️⃣ Додаткові методи

**Необхідно створити мінімум 3-4 додаткових методи:**

#### a) Метод виведення інформації
```java
public void displayInfo() {
    System.out.println("=== Інформація ===");
    System.out.println("Поле 1: " + field1);
    System.out.println("Поле 2: " + field2);
    // ...
}
```

#### b) Метод з обчисленнями або логікою
```java
public String getStatus() {
    if (someCondition) {
        return "Статус 1";
    } else {
        return "Статус 2";
    }
}
```

#### c) Метод, що змінює стан об'єкта
```java
public void updateSomething() {
    // Логіка зміни стану
}
```

#### d) Метод порівняння об'єктів
```java
public boolean compareTo(ClassName other) {
    return this.field > other.field;
}
```

---

### 6️⃣ Головний клас (Main)

**Необхідно:**
- Створити **мінімум 3 об'єкти** вашого класу
- Продемонструвати **використання всіх методів**
- Показати роботу з геттерами та сеттерами
- Додати **інтерактивне введення** даних користувачем

**Приклад:**
```java
public class Main {
    public static void main(String[] args) {
        // 1. Створення об'єктів
        Student s1 = new Student("Іван", "Петренко", 2, 85.5, "КН");
        Student s2 = new Student("Марія", "Коваль", 3, 92.0, "ІТ");
        Student s3 = new Student("Олег", "Сидоренко", 1, 67.5, "КБ");
        
        // 2. Виведення інформації
        s1.displayInfo();
        
        // 3. Використання геттерів
        System.out.println("Ім'я: " + s1.getName());
        
        // 4. Використання сеттерів
        s1.setAverageGrade(88.0);
        
        // 5. Виклик додаткових методів
        s1.promoteToNextCourse();
        
        // 6. Порівняння об'єктів
        if (s1.hasBetterGradeThan(s2)) {
            System.out.println("S1 кращий за S2");
        }
        
        // 7. Інтерактивне створення об'єкта
        Scanner scanner = new Scanner(System.in);
        System.out.print("Введіть ім'я: ");
        String name = scanner.nextLine();
        // ... створення об'єкта з введеними даними
    }
}
```
