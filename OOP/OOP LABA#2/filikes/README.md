# Лабораторна робота #2 - Основи ООП

## Опис

Ця лабораторна робота демонструє основні принципи об'єктно-орієнтованого програмування в Java:

- ✅ Створення класів та об'єктів
- ✅ Інкапсуляція даних
- ✅ Конструктори
- ✅ Геттери та сеттери
- ✅ Методи класу
- ✅ Взаємодія об'єктів

## Що реалізовано

### Клас Student
Представляє студента з такими властивостями:
- Ім'я та прізвище
- Курс навчання (1-6)
- Середній бал (0-100)
- Факультет

### Основні можливості:
1. **Створення студентів** через конструктор
2. **Отримання даних** через геттери
3. **Зміна даних** через сеттери (з валідацією)
4. **Виведення інформації** про студента
5. **Визначення статусу стипендії** на основі балу
6. **Переведення на наступний курс**
7. **Порівняння студентів** за середнім балом
8. **Інтерактивне створення** студента користувачем

## Як запустити

### В IntelliJ IDEA:
1. Відкрийте папку `LABA2`
2. IntelliJ автоматично розпізнає проект
3. Відкрийте файл `src/Main.java`
4. Натисніть зелену кнопку ▶️ біля методу `main`
5. Або натисніть `Shift + F10`

### В VS Code:
1. Відкрийте файл `src/Main.java`
2. Натисніть `Ctrl + Shift + B`
3. Виберіть "Run Java Project (IntelliJ-style)"

### Через термінал:
```bash
# Компіляція
javac -encoding UTF-8 -d bin src/Main.java

# Запуск
java -cp bin Main
```

## Що демонструє програма

### 1. Інкапсуляція
```java
private String name;  // Приватне поле
public String getName() { return name; }  // Публічний доступ
```

### 2. Конструктор
```java
public Student(String name, String surname, ...) {
    this.name = name;
    this.surname = surname;
}
```

### 3. Валідація даних
```java
public void setCourse(int course) {
    if (course >= 1 && course <= 6) {
        this.course = course;
    } else {
        System.out.println("Помилка!");
    }
}
```

### 4. Методи з логікою
```java
public String getScholarshipStatus() {
    if (averageGrade >= 90) return "Підвищена стипендія";
    else if (averageGrade >= 75) return "Звичайна стипендія";
    // ...
}
```

### 5. Взаємодія об'єктів
```java
public boolean hasBetterGradeThan(Student other) {
    return this.averageGrade > other.averageGrade;
}
```

## Основні концепції ООП

| Концепція | Опис | Приклад у коді |
|-----------|------|----------------|
| **Клас** | Шаблон для створення об'єктів | `class Student { }` |
| **Об'єкт** | Екземпляр класу | `new Student(...)` |
| **Інкапсуляція** | Приховування даних | `private String name` |
| **Конструктор** | Ініціалізація об'єкта | `public Student(...)` |
| **Геттер** | Отримання значення поля | `getName()` |
| **Сеттер** | Зміна значення поля | `setName(...)` |
| **Метод** | Поведінка об'єкта | `displayInfo()` |

## Важливі моменти

⚠️ **Не забувайте:**
- Використовувати `this.` в конструкторі
- Перевіряти дані в сеттерах
- Робити поля приватними (`private`)
- Закривати Scanner після використання

✅ **Переваги такого підходу:**
- Код організований та зрозумілий
- Дані захищені від некоректних змін
- Легко додавати нові функції
- Можна створювати багато об'єктів з одного класу
