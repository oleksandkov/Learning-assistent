# Lecture 1 - Introduction to OOP

## Topics Covered
- What is Object-Oriented Programming?
- Classes and Objects
- Basic OOP principles

## How to Run

### IntelliJ IDEA:
1. Open this folder in IntelliJ IDEA
2. Open `src/Main.java`
3. Click the green play button or press `Shift + F10`

### VS Code:
1. Open `src/Main.java`
2. Press `Ctrl + Shift + B`
3. Select "Run Java Project (IntelliJ-style)"

## Files
- `src/Main.java` - Main program with Student class example
- `note.md` - Detailed lecture notes
