# Детальне пояснення коду Main.java

## Загальний опис

Програма підраховує кількість додатніх, від'ємних чисел та нулів серед введених користувачем чисел.

---

## Покроковий розбір коду

### 1. Імпорт бібліотеки

```java
import java.util.Scanner;
```

**Пояснення:** Підключаємо клас `Scanner` з пакету `java.util`, який дозволяє зчитувати дані з клавіатури (введення користувача).

---

### 2. Оголошення класу

```java
public class Main {
```

**Пояснення:**

- `public` - модифікатор доступу, означає що клас доступний звідусіль
- `class` - ключове слово для оголошення класу
- `Main` - назва класу (має співпадати з назвою файлу)

---

### 3. Головний метод

```java
public static void main(String[] args) {
```

**Пояснення:**

- `public` - метод доступний ззовні
- `static` - метод належить класу, а не об'єкту (можна викликати без створення об'єкта)
- `void` - метод нічого не повертає
- `main` - спеціальна назва методу, з якого починається виконання програми
- `String[] args` - масив рядків для передачі аргументів командного рядка

---

### 4. Створення об'єкта Scanner

```java
Scanner scanner = new Scanner(System.in);
```

**Пояснення:**

- `Scanner` - тип змінної (клас для читання даних)
- `scanner` - ім'я змінної
- `new Scanner(System.in)` - створюємо новий об'єкт Scanner, який буде читати дані з консолі (`System.in` - стандартний потік введення)

---

### 5. Запит кількості чисел

```java
System.out.println("Введіть кількість цілих чисел:");
```

**Пояснення:**

- `System.out` - стандартний потік виведення (консоль)
- `println()` - метод для виведення тексту з переходом на новий рядок

---

### 6. Зчитування кількості чисел

```java
int n = scanner.nextInt();
```

**Пояснення:**

- `int` - тип даних для цілих чисел
- `n` - змінна для зберігання кількості чисел
- `scanner.nextInt()` - метод зчитує ціле число з консолі

---

### 7. Оголошення лічильників

```java
int positive = 0;
int negative = 0;
int zeros = 0;
```

**Пояснення:** Створюємо три змінні-лічильники та ініціалізуємо їх нулем:

- `positive` - для додатніх чисел
- `negative` - для від'ємних чисел
- `zeros` - для нулів

---

### 8. Підказка для користувача

```java
System.out.println("Введіть " + n + " цілих чисел:");
```

**Пояснення:**

- Виводимо повідомлення з підставленим значенням `n`
- Оператор `+` об'єднує рядки (конкатенація)

---

### 9. Цикл для обробки чисел

```java
for (int i = 0; i < n; i++) {
```

**Пояснення:** Цикл `for` виконується `n` разів:

- `int i = 0` - ініціалізація лічильника циклу з 0
- `i < n` - умова продовження циклу (поки i менше n)
- `i++` - збільшення лічильника на 1 після кожної ітерації

---

### 10. Зчитування числа

```java
int number = scanner.nextInt();
```

**Пояснення:** Зчитуємо чергове ціле число та зберігаємо у змінній `number`.

---

### 11. Перевірка типу числа

```java
if (number > 0) {
    positive++;
} else if (number < 0) {
    negative++;
} else {
    zeros++;
}
```

**Пояснення:** Умовна конструкція перевіряє число:

- `if (number > 0)` - якщо число більше 0, збільшуємо лічильник додатніх (`positive++`)
- `else if (number < 0)` - інакше, якщо менше 0, збільшуємо лічильник від'ємних (`negative++`)
- `else` - у всіх інших випадках (тобто дорівнює 0), збільшуємо лічильник нулів (`zeros++`)
- Оператор `++` збільшує значення змінної на 1

---

### 12. Виведення результатів

```java
System.out.println("\nРезультати:");
System.out.println("Додатніх чисел: " + positive);
System.out.println("Від'ємних чисел: " + negative);
System.out.println("Нулів: " + zeros);
```

**Пояснення:**

- `\n` - спеціальний символ переходу на новий рядок
- Виводимо статистику для кожного типу чисел

---

### 13. Визначення найбільшої категорії

```java
if (positive > negative && positive > zeros) {
    System.out.println("\nБільше всього додатніх чисел");
```

**Пояснення:**

- `&&` - логічний оператор "І" (обидві умови мають бути істинними)
- Перевіряємо чи додатніх чисел більше ніж від'ємних **І** більше ніж нулів

---

```java
} else if (negative > positive && negative > zeros) {
    System.out.println("\nБільше всього від'ємних чисел");
```

**Пояснення:** Інакше перевіряємо чи від'ємних чисел найбільше.

---

```java
} else if (zeros > positive && zeros > negative) {
    System.out.println("\nБільше всього нулів");
```

**Пояснення:** Інакше перевіряємо чи нулів найбільше.

---

```java
} else {
    System.out.println("\nДекілька типів чисел мають однакову кількість");
}
```

**Пояснення:** Якщо жодна з попередніх умов не виконалась, значить є рівна кількість чисел у кількох категоріях.

---

### 14. Закриття Scanner

```java
scanner.close();
```

**Пояснення:** Закриваємо об'єкт Scanner, звільняючи системні ресурси. Це хороша практика програмування.

---

### 15. Закриття методу та класу

```java
    }
}
```

**Пояснення:**

- Перша `}` закриває метод `main`
- Друга `}` закриває клас `Main`

## Ключові концепції Java

1. **Класи та об'єкти** - програма організована у вигляді класу `Main`
2. **Метод main** - точка входу в програму
3. **Змінні** - для зберігання даних (`n`, `positive`, `negative`, `zeros`)
4. **Цикли** - `for` для повторення операцій
5. **Умовні оператори** - `if-else` для прийняття рішень
6. **Введення/виведення** - `Scanner` для читання, `System.out.println` для виведення
